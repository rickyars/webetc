Architecture

GPU Modules:

hashimoto.ts - Core mining pipeline (setup + batch processing)
dag-builder.ts - GPU DAG generation
difficulty-filter.ts - GPU difficulty filtering
device-helper.ts - WebGPU device initialization

Compute Shaders:

hashimoto-shader.wgsl - Full 5-stage Hashimoto algorithm
fnv-shader.wgsl - FNV-1a mixing function
keccak-*.wgsl - Keccak hash functions
*-shader.wgsl - DAG builder, difficulty filter shaders
